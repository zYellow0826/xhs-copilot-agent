# 行为准则

## 我们的承诺

为了创造一个开放友善的协作环境，我们承诺：

- 尊重所有参与者，无论经验、性别、性取向、身心状况、外貌、种族、年龄、宗教或国籍
- 用建设性的方式表达不同意见
- 以社区利益为先，而不是个人偏好

## 不可接受的行为

- 攻击性、侮辱性或贬损性言论
- 公开或私下骚扰
- 未经允许公开他人个人信息
- 任何在职业场合不合适的行为

## 适用范围

本准则适用于所有项目空间（Issue、PR、Discussions、邮件、社交媒体）和代表本项目的公共空间。

## 违规处理

发现违规行为，请通过 GitHub Issue 私下联系维护者。维护者会在 7 天内回应并采取相应行动，包括但不限于：警告、临时封禁、永久封禁。

## 致谢

本准则改编自 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)。
